# Smart Spam Email Classifier

## Project Description
This project is a Machine Learning-based web application that classifies Email or SMS messages as Spam or Safe. It uses the Multinomial Naive Bayes algorithm and TF-IDF Vectorizer for text classification.

## Features
- Detects Spam and Safe messages
- Simple and user-friendly interface
- Real-time prediction
- Machine Learning based classification

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit
- TF-IDF Vectorizer

## Machine Learning Model
- Algorithm: Multinomial Naive Bayes
- Accuracy: 96.68%

## Dataset
SMS Spam Collection Dataset

## How to Run
1. Install the required libraries.
2. Run the following command:

```bash
python -m streamlit run app.py
```
