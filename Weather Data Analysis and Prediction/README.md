# 🌦️ Weather Data Analysis and Prediction

## 📌 Project Overview

Weather Data Analysis and Prediction is a Machine Learning project developed using Python and Streamlit. The application analyzes historical weather data and predicts future maximum temperature using a Linear Regression model.

Users can select a city and a historical date range to analyze weather patterns. Based on historical data, the system predicts the future maximum temperature for a selected date.

---

## 🎯 Objectives

- Analyze historical weather data.
- Visualize weather trends using graphs.
- Train a Machine Learning model.
- Predict future maximum temperature.
- Provide an interactive user interface using Streamlit.

---

## 🚀 Features

- City-wise weather analysis
- Historical date range selection
- Dataset preview
- Missing value analysis
- Statistical summary
- Temperature trend visualization
- Linear Regression model
- Future temperature prediction
- Model performance (R² Score & MAE)
- Actual vs Predicted comparison graph

---

## 📂 Dataset

Dataset Name: weather.csv

The dataset contains historical weather information including:

- Date
- City
- Maximum Temperature
- Minimum Temperature
- Rainfall
- Wind Speed
- Humidity
- Other weather-related parameters

---

## 🤖 Machine Learning Model

Algorithm Used:

- Linear Regression

Performance Metrics:

- R² Score
- Mean Absolute Error (MAE)

---

## 📊 Project Workflow

1. Load weather dataset.
2. Select city.
3. Select historical date range.
4. Analyze historical weather data.
5. Train Linear Regression model.
6. Select future prediction date.
7. Predict future maximum temperature.
8. Display graphs and model performance.

The application displays:

- Historical weather analysis

- Future maximum temperature prediction
